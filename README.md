## Overview

**Okyline** is a declarative language for describing and validating JSON documents **by example**.  
It combines readable JSON samples with inline constraints such as `|@`, `{min,max}`, `(range)`, `('enum')`, `~regex~`, `->`, `!`, and `#`.

This repository hosts the **official Okyline Language Specification** (version 1.0, November 2025).
